Foodiary UI/UX Revamp

Clean, modern, planning-first calorie tracking.

A web-based design direction proposal for Foodiary before any SwiftUI implementation. The goal is to move from the current clinical ring UI into a calmer, more polished daily planning experience.

Current app read

TodayProgress ring, planned/remaining tiles, macros, meal cards.
PlanWeek strip calendar, selected day detail, future meal planning.
InsightsMacro and calorie summary, currently lightweight.
ProfileEstimated target, BMR, maintenance, disclaimer, reset/edit.

Choose a direction

Aura

Warm modern wellness. Keeps the calorie ring mental model but makes the app feel friendlier, softer, and less clinical.

Recommendation

Use Aura as the base direction.

It preserves the current ring hero so the UX does not feel reset, but it replaces the blue clinical tone with a warmer and more premium planning experience. Borrow Zenith’s whitespace discipline to keep it clean.
